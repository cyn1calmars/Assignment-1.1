"To run Starry Toy Daycare"

Download Visual Studio Code
Download all files in the "Starry Toy Daycare codes" folder from Github
Start running and debugging "Landing Page.html" 

or

Download all files in the "Starry Toy Daycare codes" folder from Github
Go into the folder and click on "Landing Page.html"
You will be able to access any other part of the site from there.

For any concerns or questions, there is an "FAQ" page on the Home screen.
Just click the button next to the logo and it will answer any questions you have.

*Note*
Most pages when resized will not function properly or show certain aspects of the website.
